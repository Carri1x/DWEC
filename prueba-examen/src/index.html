Prueba de lanzamiento de la infraestructura para el examen.

Comando en el terminal para lanzar el servidor:

npm run servidor

Se dispondrá de los siguientes endpoint:

http://localhost:3000/usuarios

http://localhost:3000/peliculas

Para acceder a los datos de una entrada:

http://localhost:3000/usuarios/1

http://localhost:3000/peliculas/1

A partir de aquí, boton derecho sobre el fichero index.html -> Open with Live Server
